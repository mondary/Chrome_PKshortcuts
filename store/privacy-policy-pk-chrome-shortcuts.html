Politique de confidentialité

PK Chrome Shortcuts Chrome Extension

Dernière mise à jour : 26 juin 2026

Cette page est la politique de confidentialité officielle de l'extension Chrome PK Chrome Shortcuts. This page is the official privacy policy for the PK Chrome Shortcuts Chrome extension. An English version is available below the French version.

Données traitées

PK Chrome Shortcuts ne collecte pas de données personnelles pour un serveur externe. L'extension traite uniquement les données nécessaires pour fournir des raccourcis clavier et des fonctions de gestion d'onglets sur votre appareil :

  • L'onglet actif et sa fenêtre pour effectuer les actions de raccourcis ;
  • Les informations de navigation locales pour les fonctions de gestion d'onglets ;
  • Les paramètres et préférences de l'utilisateur stockés localement.

Utilisation des données

Ces données sont utilisées uniquement pour effectuer les actions de raccourcis clavier demandées par l'utilisateur et gérer les onglets dans le navigateur.

Stockage et transfert

L'extension n'envoie pas les données de navigation ou les informations utilisateur à un serveur externe. Tous les traitements sont effectués localement par Chrome sur l'appareil de l'utilisateur.

Partage et transfert

PK Chrome Shortcuts ne vend pas, ne loue pas, ne partage pas et ne transfère pas les données utilisateur à des tiers. Les données ne sont pas utilisées pour la publicité, le suivi analytique, le profilage, la solvabilité ou des finalités sans rapport avec la fonctionnalité de base de l'extension.

Cookies et suivi

L'extension n'utilise pas de cookies, de pixels de suivi, d'outils analytiques ou de technologies publicitaires.

Accès et suppression

Les paramètres et préférences sont stockés localement dans le navigateur. Les données peuvent être supprimées à tout moment en désinstallant l'extension ou en effaçant les données de navigation de Chrome.

Permissions Chrome

Les permissions déclarées servent uniquement à identifier les onglets, gérer les fenêtres, et fournir les fonctionnalités de raccourcis clavier pour la navigation et la gestion d'onglets.

Modifications de cette politique

Cette politique peut être mise à jour si les fonctionnalités de l'extension changent. La date de mise à jour sera modifiée sur cette page.

Contact

Pour toute question concernant cette politique de confidentialité, contactez : contact@mondary.design.

English Version

Data Processed

PK Chrome Shortcuts does not collect personal data for an external server. The extension only processes the information required to provide keyboard shortcuts and tab management functionality on your device:

  • The active tab and its window to perform shortcut actions;
  • Local browsing information for tab management functions;
  • User settings and preferences stored locally.

Use of Data

This data is used only to perform the keyboard shortcut actions requested by the user and manage tabs in the browser.

Storage and Transfer

The extension does not send browsing data or user information to an external server. All processing is done locally by Chrome on the user's device.

Sharing and Transfer

PK Chrome Shortcuts does not sell, rent, share, or transfer user data to third parties. Data is not used for advertising, analytics tracking, profiling, creditworthiness, or purposes unrelated to the extension's core functionality.

Cookies and Tracking

The extension does not use cookies, tracking pixels, analytics tools, or advertising technologies.

Access and Deletion

Settings and preferences are stored locally in the browser. Data can be deleted at any time by uninstalling the extension or clearing Chrome browsing data.

Chrome Permissions

The declared permissions are used only to identify tabs, manage windows, and provide keyboard shortcut functionality for navigation and tab management.

Changes to This Policy

This policy may be updated if the extension features change. The update date will be changed on this page.

Contact

For questions about this privacy policy, contact: contact@mondary.design.
